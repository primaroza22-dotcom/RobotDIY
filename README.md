# RobotDIY

## Open Source DIY AI Robot Platform

RobotDIY adalah proyek open-source untuk membangun robot AI modular yang dapat dicetak dengan printer 3D.

## Target Platform

- ESP32-S3
- Raspberry Pi 5
- Home Assistant
- ROS2
- YOLO
- OpenCV
- FreeCAD
- Bambu Studio

## Sprint

- Sprint-001 : Chassis + Motor + ESP32
- Sprint-002 : Camera Module
- Sprint-003 : Streaming
- Sprint-004 : Home Assistant
- Sprint-005 : AI Vision
- Sprint-006 : Face Recognition
- Sprint-007 : Auto Docking
- Sprint-008 : SLAM
- Sprint-009 : Voice Assistant
- Sprint-010 : Full AI Robot
